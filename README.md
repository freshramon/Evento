# Fusión Gourmet - Festival Gastronómico de Vanguardia

## Project Description

This is a landing page for "Fusión Gourmet", a gastronomic festival that takes place in Bogotá, Colombia. The page shows the festival's schedule, the invited chefs, a photo gallery and contact information.

I built this project as a practice exercise to apply what I've learned about HTML5 and CSS3.

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Playfair Display and Poppins)

## Project Structure

```
fusion-gourmet/
│
├── index.html     (main page with all the content)
├── styles.css     (all the styles)
└── README.md      (this file)
```

### Sections in index.html

- **Header**: logo and navigation menu
- **Hero**: main section with a call to action
- **Schedule**: program table and featured event cards
- **About**: info about the festival, values list and chef cards
- **Gallery**: photo grid and simulated video
- **Sponsors**: list of main sponsors
- **Footer**: contact info and social media links

## How to View the Project

1. Download or clone the files
2. Open the `index.html` file directly in your browser
3. That's it, no installation needed

You can also use the Live Server extension in VS Code if you have it installed.

## Notes

- I used placeholder gradients instead of real images since I didn't have photos yet
- The layout uses Flexbox and CSS Grid
- There are media queries for mobile (480px) and tablet (768px and 1024px)
